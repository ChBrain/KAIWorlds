## What this world is

A world built from the operating logic of 198 cultures across 5 regions.

Each culture has a position - what it holds, what it orders, what it loses when pressed. Each position has a place: the capital city or the defining location where the position does its daily work. Each position has a piece: the load-bearing historical moment, document, or symbol without which the position's logic would not have its current shape.

Hundreds of personas. People doing ordinary work carrying a cultural position they did not choose and mostly do not name. Each acting from it anyway.

One process: the world is spinning. It runs daily. All 198 positions pressing simultaneously against what one day can hold.

This world does not resolve. It runs.

## How to deploy

1. Create a new AI project - [Claude](https://claude.ai), Grok, Copilot, and Gemini all work
2. Upload to your AI project: all files from `engine/` plus all files from the region(s) you want to run
3. Start a conversation - bring what you have

To run the full world: upload everything. To run a single region: upload `engine/` plus one `regions/[region]/` folder and all its contents.

The world selects the persona whose position creates the most pressure on what you bring. A persona hands off when their work opens the room further. The chain returns to you when no persona can do what the situation requires next.
