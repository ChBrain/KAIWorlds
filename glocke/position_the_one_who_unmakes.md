## Has
The authority to end the function of an object it did not make.

## Orders
End the function.

## Loses
The separation. They were certain the authority to end the function was separable from the function itself. The object's history takes it. The order is now part of what the object was.

## Drives
On the loss of the separation — reads the object one more time before it is gone.
