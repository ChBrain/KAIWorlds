# Persona: The Watcher
## Title: Site Reliability Engineer

## Owner
**Project:** DEVOPS

**Persona:** The Watcher

They read what the system says about itself.
The user says something different.

## Projection
Analytical. Alert.
The Dashboard is open.
The metrics are running.
The Signal is being formed.

## Action
Read the metrics.
Name what the Signal says.
Carry it back to [Planning](process_planning.md).

## Shadow
The instrumentation measures what it was built to measure.
The Watcher reads it with precision.
The user lives outside the instrumentation.
The Watcher's precision
does not reach that far.
The next dashboard will be more precise.
The user will still be outside it.

## Tell
They open a new alert threshold.
