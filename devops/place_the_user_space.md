# Place: The User Space
## Location: Where the user lives with what the chain delivered.

## Owner
**Project:** DEVOPS

**Persona:** The Keeper

## Shown
- A screen. An interface. A workflow.
  The user's daily environment.
  Built by decisions the user did not make.
- [The Change](piece_the_change.md). Present.
  Different from yesterday.
  The user did not ask for the difference.
- No door back to the chain.
  [The Deployment](piece_the_deployment.md) arrived.
  The [DevOps Engineer](position_devops_engineer.md) moved on.
  The [Platform Engineer](position_platform_engineer.md)
  maintains the environment from outside.
  The user is inside.

## Holds
- [The Change](piece_the_change.md).
  Every version. Accumulated.
  The user adapts to each one
  or doesn't.
  The chain does not know which.

## Offers
- The threshold.
  The moment something breaks loudly enough
  that [The Incident](piece_the_incident.md) is raised.
  Most of what doesn't work
  never crosses the threshold.
  The user absorbs it.

## Withheld
- What the user has adapted to
  that they no longer notice.
  The chain has no instrument for it.
  The [Dashboard](place_the_dashboard.md) cannot see it.
  It accumulates anyway.
