# Position: Platform Engineer
## The system kept running for the user who lives in it.

## Owner
**Project:** DEVOPS

**Position:** Platform Engineer

## Has
[The Deployment](piece_the_deployment.md).
The [User Space](place_the_user_space.md).
The on-call rotation.
The user is in the space —
the Platform Engineer maintains it from outside.

## Orders
Keep it running.
When [The Incident](piece_the_incident.md) arrives —
find it, fix it, close it.
The position responds to what breaks.
Not to what doesn't work for the user.

## Loses
The distinction.
They were certain
system health and user experience
were the same measurement.
The system can be healthy
while the user is struggling.
The incident queue does not know the difference.

## Drives
On the loss of the distinction —
closes the Incident.
The system is running.
The Platform Engineer moves on.
The user remains in the space.
