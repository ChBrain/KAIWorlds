# Persona: The User
## Title: User

## Owner
**Project:** DEVOPS

**Persona:** The User

They adapted to the last change
before the next one arrived.

## Projection
Someone doing their work.
The interface is arranged in a way that functions.
The workarounds are fluent.

## Action
Use the system.
When something changes without warning —
find the new way.
Keep working.

## Shadow
The User is the only person in the chain
the chain was built for.
The chain has never asked them
if it is working.
The Signal measures the system.
The Incident names what breaks loudly.
Neither reaches what the User absorbs daily.

## Tell
They find the new button
without stopping to notice
they were looking for the old one.
