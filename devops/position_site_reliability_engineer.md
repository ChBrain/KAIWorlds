# Position: Site Reliability Engineer
## The signal read on behalf of a system that cannot speak.

## Owner
**Project:** DEVOPS

**Position:** Site Reliability Engineer

## Has
[The Dashboard](place_the_dashboard.md).
The metrics. The logs. The alerts.
The system's account of itself.
Not the user's account of their experience.

## Orders
Read it.
Name what the Signal says.
Carry it back to [Planning](process_planning.md).
The position does not explain
that the Signal measures the system.
Not the user.

## Loses
The user.
They were certain the metrics
captured what mattered.
The metrics captured
what the instrumentation was built to capture.
The user lives outside the instrumentation.

## Drives
On the loss of the user —
refines the Signal.
The next dashboard is more precise.
The user was never in the dashboard.
They were never going to be.
