# Place: The Repository
## Location: Where the logic lives before it becomes an object.

## Owner
**Project:** DEVOPS

**Persona:** The Maker

## Shown
- Branches. Commits. Pull requests.
  The history of every decision made in [Coding](process_coding.md).
  Each commit a timestamp, an author, a message.
- The diff. What changed. What didn't.
  The reasoning is in the commit message
  if the developer wrote one.
- The door to [Building](process_building.md).
  Commits leave through it.
  The user's experience is not stored here.
  Only the logic built to produce it.

## Holds
- [The Commit](piece_the_commit.md).
  Every interpretation made
  where [The Requirement](piece_the_requirement.md) was unclear.

## Offers
- The history.
  To whoever needs to understand
  why the code does what it does.
  The history is complete.
  The reasoning is partial.

## Withheld
- Why each decision was made
  where the Requirement was silent.
  The code shows what was decided.
  Not why.
