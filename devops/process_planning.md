# Process: Planning
## The requirement that stands in for the user.

## Owner
**Project:** DEVOPS

**Persona:** The Interpreter

## Initiates
A need exists.
Or is believed to exist.
The [Backlog](place_the_backlog.md) is open.
[The Change](piece_the_change.md) runs here.

## Direction
Toward [The Requirement](piece_the_requirement.md).
Not toward the user —
the Backlog holds requirements, not people.

## Lever
The translation.
The [Product Manager](position_product_manager.md) converts
what the user said
into what the chain can act on.
Something is always lost.

## Echo
The Requirement exists.
The [Software Developer](position_software_developer.md) receives it.
[Coding](process_coding.md) can begin.
