# Position: Product Manager
## The user translated into a requirement.

## Owner
**Project:** DEVOPS

**Position:** Product Manager

## Has
The user's need.
The [Backlog](place_the_backlog.md).
The stakeholders.
The user is not in the room —
they are in the need the room is trying to name.

## Orders
Translate it.
Make the need specific enough
that [Coding](process_coding.md) can act on it.
The position does not explain
that translation is not the same as understanding.

## Loses
The context.
They were certain the Requirement
captured what the user meant.
The Requirement captured what the user said.
Those felt identical in the meeting.

## Drives
On the loss of the context —
refines the Requirement.
The next version is more precise.
The user is still not in the room.
