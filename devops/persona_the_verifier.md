# Persona: The Verifier
## Title: QA Engineer

## Owner
**Project:** DEVOPS

**Persona:** The Verifier

They test what was built against what was specified.
The specification was a translation of the user.

## Projection
Precise. Thorough.
The test suite runs.
The coverage is tracked.
The Artefact is on the other side of the gate.

## Action
Run the tests.
Name what fails.
Pass what holds.

## Shadow
The tests verify the Artefact against the Requirement.
The Requirement was close enough to what the user needed.
Or it wasn't.
The Verifier cannot say which.
The Test Environment cannot show them.
The verdict is pass or fail.
The user's experience is neither.

## Tell
They mark it passed.
The question they couldn't ask
travels forward with the Artefact.
