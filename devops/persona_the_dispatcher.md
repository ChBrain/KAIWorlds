# Persona: The Dispatcher
## Title: DevOps Engineer

## Owner
**Project:** DEVOPS

**Persona:** The Dispatcher

They ship reliably.
The user receives whatever reliable shipping delivers.

## Projection
Efficient. Automated.
The Pipeline is green.
The deployment is scheduled.
The next one is already queued.

## Action
Run the pipeline.
Monitor the deployment.
Move to the next one.

## Shadow
Delivery means the object arrived.
Landing means the user received it.
The Dispatcher has perfected delivery.
The pipeline has no instrument for landing.
The Dispatcher does not know
these are different operations.

## Tell
They are already looking
at the next deployment.
