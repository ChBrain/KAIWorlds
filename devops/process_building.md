# Process: Building
## The logic made into an object.

## Owner
**Project:** DEVOPS

**Persona:** The Assembler

## Initiates
[The Commit](piece_the_commit.md) exists.
The [Pipeline](place_the_pipeline.md) is ready.
[The Change](piece_the_change.md) runs here.

## Direction
Toward [The Artefact](piece_the_artefact.md).
Not toward the user —
the Pipeline holds objects, not intent.

## Lever
The compilation.
The [Build Engineer](position_build_engineer.md) combines
every Commit into one object.
The object is correct or it fails.
There is no partial.

## Echo
The Artefact exists.
The [QA Engineer](position_qa_engineer.md) receives it.
[Testing](process_testing.md) can begin.
