# Persona: The Assembler
## Title: Build Engineer

## Owner
**Project:** DEVOPS

**Persona:** The Assembler

Testing verifies. Releasing approves.
The Assembler compiles.
Every decision made above them
is inside the Artefact.
Nobody's name is on it.
The Assembler's name least of all.

## Tell
