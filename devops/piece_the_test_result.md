# Piece: The Test Result
## The object's verdict on itself.

## Owner
**Project:** DEVOPS

**Persona:** The Verifier

## Place
[The Test Environment](place_the_test_environment.md).

## Load Bearing
Without it the chain has no gate.
With it the chain has confidence
that the object does what it was specified to do.
The specification was a translation of the user.
The Test Result does not know this.

## Apparent
Pass or fail.
A report. A percentage.
Green or red.
It looks like quality assurance.
It is assurance that the object matches its description.
Not that the description matched the user.

## Yearbook
Produced in [Testing](process_testing.md).
The [QA Engineer](position_qa_engineer.md) ran the tests.
The tests passed.
The [Artefact](piece_the_artefact.md) does what
[The Requirement](piece_the_requirement.md) said it should.
The Requirement was close enough to what the user needed.
Or it wasn't.
The Test Result cannot say which.
