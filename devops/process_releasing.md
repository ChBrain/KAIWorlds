# Process: Releasing
## The object approved for the world.

## Owner
**Project:** DEVOPS

**Persona:** The Gatekeeper

## Initiates
[The Test Result](piece_the_test_result.md) exists.
The [Staging Environment](place_the_staging_environment.md) is ready.
[The Change](piece_the_change.md) runs here.
The Test Result stands in for [The User Space](place_the_user_space.md).
User readiness is not in the room.

## Direction
Toward [The Release Note](piece_the_release_note.md).
Not toward the Deployment —
toward the approval that makes it possible.

## Lever
The sign-off.
The [Release Manager](position_release_manager.md) approves
what passed the test
for the world it has never been in.
The Staging Environment resembles production.
It is not production.

## Echo
The Release Note exists.
The [DevOps Engineer](position_devops_engineer.md) receives it.
[Deploying](process_deploying.md) can begin.
