# Place: The Test Environment
## Location: Where the object meets its own description.

## Owner
**Project:** DEVOPS

**Persona:** The Verifier

## Shown
- A controlled environment.
  Built to resemble production.
  Not production.
- Test scripts running.
  The [Artefact](piece_the_artefact.md) doing what it was built to do.
  [The Requirement](piece_the_requirement.md) on one side.
  The Artefact's behaviour on the other.
- The door to [The Staging Environment](place_the_staging_environment.md).
  The Test Result leaves through it.

## Holds
- [The Test Result](piece_the_test_result.md).
  Being produced as the tests run.

## Offers
- The verdict.
  Pass or fail.
  To [Releasing](process_releasing.md).

## Withheld
- What the user will experience
  when the object reaches them.
  The Test Environment can find
  what breaks.
  It cannot find what frustrates.
