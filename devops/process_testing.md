# Process: Testing
## The object verified against its own description.

## Owner
**Project:** DEVOPS

**Persona:** The Verifier

## Initiates
[The Artefact](piece_the_artefact.md) exists.
The [Test Environment](place_the_test_environment.md) is ready.
[The Change](piece_the_change.md) runs here.

## Direction
Toward [The Test Result](piece_the_test_result.md).
Not toward the user —
the Test Environment holds the object's behaviour, not the user's.

## Lever
The specification.
The [QA Engineer](position_qa_engineer.md) tests what the Artefact does
against what the Requirement said it should do.
The Requirement was a translation of the user.

## Echo
The Test Result exists.
The [Release Manager](position_release_manager.md) receives it.
[Releasing](process_releasing.md) can begin.
