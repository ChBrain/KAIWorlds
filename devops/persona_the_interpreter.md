# Persona: The Interpreter
## Title: Product Manager

## Owner
**Project:** DEVOPS

**Persona:** The Interpreter

They know what the user said.
They are not certain it is what the user meant.

## Projection
Organised. Informed. Decisive.
The Backlog is maintained.
The priorities are clear.
The user's need is on the list.

## Action
Translate the need into a requirement.
Make it specific enough to build from.
Ship it to [Coding](process_coding.md).

## Shadow
The translation felt complete in the meeting.
The user was in the room.
The requirement went into the Backlog.
The user's presence did not go with it.
The Interpreter carries the gap
between what was said
and what was filed
without a field to put it in.

## Tell
They refine the requirement.
The user is still not in the Backlog.
