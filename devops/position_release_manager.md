# Position: Release Manager
## The approval that opens the door to the world.

## Owner
**Project:** DEVOPS

**Position:** Release Manager

## Has
[The Test Result](piece_the_test_result.md).
[The Release Note](piece_the_release_note.md).
The authority to sign.
The Staging Environment is not the world.

## Orders
Approve it.
The Test Result passed.
Sign.
The position does not ask
what the user will find
on the other side of the signature.

## Loses
The preview.
They were certain Staging
had shown them what production would be.
Staging showed them what the object does.
Not what it does to the user.

## Drives
On the loss of the preview —
signs [The Release Note](piece_the_release_note.md).
[The Change](piece_the_change.md) is approved.
