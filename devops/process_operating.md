# Process: Operating
## The world running on the object.

## Owner
**Project:** DEVOPS

**Persona:** The Keeper

## Initiates
[The Deployment](piece_the_deployment.md) exists.
The object is live in [The User Space](place_the_user_space.md).
The user is in the room.
[The Change](piece_the_change.md) runs here.
The Incident stands in for [The User Space](place_the_user_space.md).
What the user absorbs is not in the room.

## Direction
Toward continuation.
Not toward the user —
toward keeping the object running
in the space the user inhabits.

## Lever
The incident.
The [Platform Engineer](position_platform_engineer.md) responds
to what breaks.
Not to what doesn't work for the user.
To what stops working for the system.
Those are not the same thing.

## Echo
The object continues running.
Or it doesn't.
[The Incident](piece_the_incident.md) is raised.
The [Site Reliability Engineer](position_site_reliability_engineer.md) receives it.
[Monitoring](process_monitoring.md) can begin.
