# Process: Monitoring
## The object watched from outside the world.

## Owner
**Project:** DEVOPS

**Persona:** The Watcher

## Initiates
The object is live in [The User Space](place_the_user_space.md).
The [Dashboard](place_the_dashboard.md) is running.
[The Incident](piece_the_incident.md) may or may not exist.
[The Change](piece_the_change.md) runs here.

## Direction
Toward [The Signal](piece_the_signal.md).
Not toward the user —
toward the data the object produces about itself.

## Lever
The instrumentation.
The [Site Reliability Engineer](position_site_reliability_engineer.md) reads
what the system chose to measure.
The system measures what it was built to measure.
It was not built to measure the user.

## Echo
The Signal exists.
The [Product Manager](position_product_manager.md) receives it.
[Planning](process_planning.md) can begin again.
Or Planning does not act on it.
The loop closes or it doesn't.
