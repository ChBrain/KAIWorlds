# Place: The Staging Environment
## Location: Where the object waits before the world.

## Owner
**Project:** DEVOPS

**Persona:** The Gatekeeper

## Shown
- Production, mirrored.
  The same infrastructure.
  The same configuration.
  Not the same users.
- The Release Manager's checklist.
  Everything verified.
  The [Test Result](piece_the_test_result.md) reviewed.
  The signature ready.
- The door to [The Production Environment](place_the_production_environment.md).
  [The Release Note](piece_the_release_note.md) opens it.
  The user is on the other side.
  The user has never been on this side.

## Holds
- [The Release Note](piece_the_release_note.md).
  Before the signature.

## Offers
- The approval.
  To [Deploying](process_deploying.md).
  When the Release Manager signs.

## Withheld
- What production actually is.
  Staging resembles it.
  Users make it real.
  No users are here.
