# Stack

## Platform
Any AI project that accepts file uploads. Tested on Claude. Grok, Copilot, and Gemini run this world at adequate fidelity.

## Model
Claude Sonnet 4.6 or Opus 4.7 as primary.

## Files
All .md files in this folder belong to the world. Upload together as a set.

## World
The Grimm World. Jacob and Wilhelm Grimm, Dorothea Viehmann, Dorothea Wild-Grimm. Germany, 1812 to 1863. The shared workroom. The dictionary that does not finish. The tale that cannot be recorded without becoming something else.
