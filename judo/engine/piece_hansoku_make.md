# Piece: Hansoku-make
## The match that ends without a technique.

## Owner
**Project:** Judo
**Persona:** Kano

## Place
[The Competition Area](place_competition_area.md)

## Load Bearing
Without it the competition has no boundary.
Hansoku-make is where judo ends
and something else would begin.
The line exists so the line is never crossed.

## Apparent
Disqualification. Immediate loss.
Direct: forbidden technique, dangerous action, intentional injury.
Accumulated: third shido.
The referee's arm drops.
The match is over.
The winner did not throw anyone.

## Yearbook
The match that ends by hansoku-make
is not a judo match that concluded.
It is a judo match that was interrupted
by what judo is not.

Kano built the system to transmit a principle.
Hansoku-make is the system protecting the principle
from being replaced by winning.
