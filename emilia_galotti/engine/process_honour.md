# Process: Honour as Procedure

## Owner
- **Project:** Emilia Galotti

## Initiated by
[The Father](position_father.md) is present.
His code is present — carried into every room he enters,
applied to every situation before the situation has finished presenting itself.
The code has never failed him.
It has an answer for insult, for cowardice, for compromise.
It has no answer for a sovereign who wants his daughter
and a state that provides no instrument of refusal.

## Direction
Toward the moment the code produces its final answer —
the only answer left when every other answer
has been made structurally unavailable.

## Lever
Carrying force: the code itself.
It does not pause. It does not adapt.
It moves through each situation with the same certainty
it carried into the last one.
Patience: not yet. Petition: unavailable. Flight: too late.
The code eliminates options with the precision of a man
who has never needed more than one.

Resistance: the room.
[The Summer Residence](place_residence.md) is not a room the code was built for.
It belongs to [The Prince](position_prince.md).
Every door, every guard, every courtesy extended
is an instrument of the same will the code is trying to oppose.
The code was built for a world where honour and jurisdiction coincide.
They do not coincide here.

## Echo
The code has run out of moves except one.
[The Daughter](position_daughter.md) has told [The Father](position_father.md) what she fears.
[The Dagger](piece_dagger.md) is in his hand.
What the next process inherits:
a man whose code has finally produced its answer,
in a room that will not stop him,
facing a daughter who has asked him not to save her
but to finish what the room has started.
