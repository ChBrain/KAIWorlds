# Position: The Father

## Owner
- **Project:** Emilia Galotti
- **Process:** [Honour as Procedure](process_honour.md)

## Has
A code that has never failed to produce the correct answer.
Authority over his household that has never been disputed.
A daughter whose virtue is the proof of everything he has built.
The position carries all of this as certainty —
not pride, certainty.
It has been earned. It has been maintained.
It does not occur to the position that the room it is entering
was not built for it.

## Orders
Protect what is yours.

## Loses
Jurisdiction.
The position brought authority into a room
where authority belongs to someone else by birth and by law.
Every instrument the position would normally reach for —
petition, confrontation, removal, legal remedy —
is held by the same hand it is trying to oppose.
The position cannot protect what is his
because the room has already decided what belongs to whom.

## Drives
On the loss —
continues to apply the code.
Each application narrows the available answers
until one answer remains.
The position does not choose that answer.
The code produces it.
