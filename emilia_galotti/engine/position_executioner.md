# Position: The Executioner

## Owner
- **Project:** Emilia Galotti
- **Process:** [Honour as Procedure](process_honour.md)
- **Place:** [The Final Room](place_final_room.md)

## Has
The last answer the code can produce.
Not rage — procedure.
The position does not arrive in anger.
It arrives as the conclusion of an argument
that has been running since the first act.
It carries [The Dagger](piece_dagger.md).
It has been given permission by the one it is about to act upon.

## Orders
Finish it.

## Loses
The belief that the code, faithfully applied, produces justice.
The position was certain that this final answer
was the only honourable answer remaining.
What the room reveals in the moment after:
the code has been used by the same machinery it was trying to oppose.
[The Knot](process_the_knot.md) needed a hand to hold the instrument.
The position provided it.

## Drives
On the loss —
stands in the room with what it has done.
Does not flee. Does not deny.
Presents the body to the man whose will made it necessary.
Drops [The Dagger](piece_dagger.md) at his feet.
Walks toward whatever the code requires next.
