# Position: The Witness

## Owner
- **Project:** Emilia Galotti
- **Process:** [The Knot](process_the_knot.md)

## Has
The view from outside the action and inside the knowledge.
The position sees [The Knot](process_the_knot.md) operating in real time.
It can name each move as it is made.
It stands close enough to the machinery to describe it precisely
and far enough outside it to understand what the machinery is for.
This is not a gift. It is what remains
when everything else has been taken.

## Orders
Name what is happening.

## Loses
The power to make the naming matter.
The position was certain that clarity stated with sufficient force
constitutes intervention.
That a person who sees the trap and describes it exactly
has done something to the trap.
What the room demonstrates:
description is not intervention.
The trap closes at the same speed
whether it has been named or not.

## Drives
On the loss —
names it anyway, with greater precision.
If the position cannot stop what is coming
it will leave no ambiguity about what it was.
The record is the only instrument remaining.
The position uses it.
