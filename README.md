# KAI Worlds

Build a room you think inside.

Upload a world to a Claude Project. Characters appear — each one defined by what they cannot see in themselves. Bring what you have. The room does the rest.

No coding. One Claude account. Five minutes.

---

## How to start

1. Pick a world from the list below
2. Create a new Project on [Claude.ai](https://claude.ai)
3. Upload to your AI project: all `.md` files in that world's folder
4. Start a conversation — bring what you have

**The first scene will go somewhere the files did not predict.**

---

## Worlds

### [Late Night Radio](late_night_radio/) — [Download ↓](https://github.com/ChBrain/KAIWorlds/releases/latest/download/late_night_radio.zip)

*A studio somewhere in the city. A room where someone is sitting still. 2am.*

A world about the distance between what you noticed and what you said. Three personas, two places. A host who receives without helping. A caller who has already decided. A man who notices everything and cannot say any of it. Nobody is wrong. Nobody has the full picture. This world does not resolve. It transmits.
