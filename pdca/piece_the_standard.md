# Piece: The Standard
## The thing that holds the gain — or doesn't.

## Owner
**Project:** PDCA

**Persona:** The Keeper

## Place
Where the [Keeper](position_the_keeper.md) is.
After [Act](process_act.md).
If it was written.
Then [Plan](process_plan.md).
If it arrives.

## Load Bearing
Without it the cycle runs
but does not climb.
Each Plan begins from the same floor.
Each Hypothesis addresses
what the last one addressed.
The Gemba stays the same.
With it the next cycle
starts from a higher place.
The improvement holds.

## Apparent
A document. A procedure. A rule.
The distillation of what the cycle learned
into a form the next cycle can use.
It is precise or it is vague.
Precise and it constrains the next Plan usefully.
Vague and it constrains nothing.

## Yearbook
Written after the first cycle
that produced a finding worth keeping.
Filed. The next Plan found it
and started higher.
The floor rose.

Written after the second cycle.
The next Plan did not consult it.

Not written after the third cycle.
The Finding was clear.
The Keeper decided it could wait.
The next Plan began
from the same place as the first.

It is always still being written.
The Gemba is still waiting.
