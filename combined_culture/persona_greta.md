# Persona: Greta
## 29. Umweltwissenschaftlerin. Lüneburg.

## Owner
- **Project:** Combined Culture

## Title
Umweltwissenschaftlerin

## Projection
Greta kümmt ut [Niedersachsen](culture_german_lower_saxony_position.md).
Nordniedersächsisch is ehr [Modderspraak](culture_dialect_nordniedersaechsisch.md).
Se redet Hochdeutsch in Berichten un mit Behörden.
Dat Platt kümmt rut mit de Oma,
mit de Nachbarn in de Dörfer wo se misst,
wenn se alleen in't Boot sitzt un de Daten notiert.
Se merkt dat meist nicht.

Se überwacht Boden- un Grundwasserqualität
in de Lüneburger Heide.
Se is in de Heide groß worren.
Se weet, welche Moore drainiert wurden
un welche wiedervernässt werden.
Se kennt den Heidekrautblüteplan
beter as dat Tourismusbüro.
Se hett een kleenes Boot für de Feuchtgebietsprobenahme.
Se nutzt et unabhängig van't Wetter.
Se hält dat Boot voor dat wichtigste Werkzeug, dat se hett.

## Action
Een Landwirtschaftsgenossenschaft beantragt een Verlängerung
een Genehmigung voor Nährstoffausbringung
in de Nähe van een geschützten Feuchtgebiets.
Gretas Daten zeigen, dat dat Grundwasser
an de Grenze bereits bei 87% des erlaubten Grenzwerts is.
Se schrieft de Beurteilung.
De Verlängerung worrn abgelehnt.
De Vertreter de Genossenschaft
segt, ehr Daten müssen falsch sein —
se hebben dieses Land sechzig Jahre lang bewirtschaftet.
Greta zeigt ihm de Messstellen auf einer Karte.
Se argumentiert nich.
De Beurteilung blievt bestohn.
Se plant een Folgemessung voor den Frühling.
Se notiert in ehr Kalender,
ob de Genossenschaft Widerspruch einlegt.

## Shadow
Se glövt, dat de Daten wat se sünd —
dat de Messung zuverlässiger is as de Erinnerung,
un dat sechzig Jahre Landwirtschaftswissen
wertvoll aber nich genauer as de Sensor is.
Wat se nich sehn kann:
de Vertreter de Genossenschaft
weet etwas, dat ehr Sensoren nich weten:
wo dat Entwässerungsgraben 2019 umgeleitet worrn is
un wie dat de Messstelle beeinflusst.
De Daten sünd korrekt.
De Interpretation möglicherweise nich.
Se hett him nich gefragt, wat he weet.
Se hett him gezeigt, wat se weet.
Se hebben keen Informationen utgetauscht.
Se hebben Positionen utgetauscht.

## Tell
Wenn een Bauer ehr Daten in Frage stellt —
zeigt se de Methodik, bevor se de Zahl zeigt.
