# Work: Boundary Worker

## Owner
- Project: Combined Culture

## Has
Two languages. Two filing systems. Two calendars. The position operates inside two formal systems simultaneously, each of which assumes it is the only system. The person knows both assume this. The position tells neither.

## Orders
Complete the form that does not describe the situation. Write the explanatory note. File it. Move on.

## Loses
Legibility. The work is real. The work is documented only in files nobody reads. What the position actually does — the silent translation between two systems — exists in no form. It has no name. It has no evaluation column. It is not missed when absent because it was never explicitly requested.

## Drives
On the loss of legibility —
navigates.
The position has not yet decided what to do with the unreadability.
It navigates while not deciding.
The navigation becomes the decision
by deferring it.
Ten years pass.
Twenty.
The form still asks for a single origin.
The person still files the note.
